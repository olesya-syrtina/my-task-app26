# Kanban Board

Мини-приложение для управления задачами в формате Kanban-доски, реализованное на React и TypeScript.

## Функциональность:

Отображение колонок (Todo, In Progress, Done)
Добавление новых задач
Список задач внутри каждой колонки
Обновление состояния без мутаций (immutable подход)

## Установка и запуск

```bash
git clone https://github.com/your-username/.git
npm install
npm run dev
